
# Personal Developer Portfolio

A modern, responsive portfolio website built to showcase my skills, projects, internships, and achievements as a Full Stack Developer. Developed using HTML5, CSS3, and JavaScript with a clean UI, smooth animations, and optimized performance.

## Features

- Modern and fully responsive design
- Dark & Light mode with saved user preference
- Smooth scrolling and interactive animations
- Project filtering by category
- Skills, experience, education, and certification sections
- Contact form with validation
- Optimized for performance and accessibility

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- Font Awesome
- Google Fonts
